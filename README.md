# To-Do List App

**Author:** Mackenzie Chaney

## Description

A Java to-do list application with a Swing GUI. The app allows users to add, remove, and edit items on a to-do list. The list updates automatically after every change.

## Project Structure

* **TodoList.java** – handles all list logic 
* **TodoAppGUI.java** – the Swing GUI that the user can interact with using buttons

## How to Use

1. Run the GUI file

2\. Use the buttons to manage your list:

* **Add** – type an item in the text field at the top and click Add
* **Remove** – click Remove and enter the number of the item to delete
* **Edit** – click Edit, enter the item number, then enter the new text
